# ChronoStride-Unified-Gait-Analysis-Framework
An intelligent deep learning-based system for analyzing human gait patterns for **healthcare diagnosis** and **forensic identification**.

## Overview
This project uses advanced AI techniques to recognize individuals and detect neurological disorders based on walking patterns. It combines spatial and temporal analysis for high accuracy in real-world conditions. 

## Features
* Gait-based **person identification**
* Detection of **Parkinson’s** and **Alzheimer’s**
* **Healthcare + Forensic dual module**
* Real-time **video gait analysis**
* Automated **report generation**

## Technologies Used
* Machine Learning & Deep Learning
* CNN, LSTM, CNN-LSTM Hybrid
* Graph Transformer Networks
* Self-Supervised Learning
* Computer Vision & Image Processing 

## System Architecture
* Input: Video / Sensor Data
* Preprocessing: Silhouette & Pose Extraction
* Feature Extraction: Spatial + Temporal Features
* Model: CNN-LSTM & Transformer
* Output: Disease Prediction / Suspect Identification 

## Results
* **86.7% accuracy** in healthcare disease detection
* **91.2% accuracy** in forensic identification 

## Dataset
* Healthcare: PhysioNet gait dataset
* Forensic: CASIA-B gait dataset 

## Future Work
* Real-time deployment
* Multi-modal data integration
